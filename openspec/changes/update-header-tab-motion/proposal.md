# Proposal: 调整 Header Tab 动效与对齐

## Why

当前 header 的主切换区还有三个问题：

1. `学习` / `错题本` 切换时，用户感知主要来自整页内容切换，Tab 自身缺少明确的局部反馈
2. `Linux 命令行学习平台` 标题文案占位明显，但信息价值低
3. Tab 组当前位置偏中，不符合这两个核心入口应靠左的预期

## What Changes

1. **Tab 局部滑块动画**
   - 只在 `学习` / `错题本` 两个 Tab 内部增加切换动画
   - 当前激活 Tab 的背景块在两个标签之间横向滑动
   - 不引入整页级别的新切换动画

2. **移除 Header 标题文案**
   - 删除 `Linux 命令行学习平台` 这行文字

3. **Tab 组左移**
   - 将 `学习` / `错题本` 这组标签移动到 header 左侧区域
   - 不再保持当前近似居中的位置

## Impact

- **影响文件**：`frontend/src/App.tsx`
- **可能补充样式**：`frontend/src/index.css`
- **无业务逻辑变更**：只调整 header 结构和 tab 视觉反馈
